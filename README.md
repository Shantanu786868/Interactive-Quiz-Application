🛠️ Technologies Used
HTML5: Used to build the structural foundation of the application, including the containers for questions, multiple-choice buttons, and the final score display.

CSS3: Used for styling the user interface. This includes layout design, typography, color schemes, and interactive visual feedback (like hover effects and highlighting correct/incorrect answers).

JavaScript: Used to handle the entire logic and interactivity of the application without relying on external libraries or frameworks.

Event Handling: JavaScript event listeners are attached to buttons (e.g., clicking an answer, clicking the "Next" button) to trigger the appropriate functions.

Results Display: A dedicated end-screen that calculates the final percentage or total points and presents the user with their final score.

Dynamic Data Rendering: Questions, options, and correct answers are stored in JavaScript (typically as an array of objects) and injected dynamically into the HTML using DOM manipulation.

AI & Prototyping Tools: * Bolt: Used for AI-driven code generation and rapid scaffolding.

Builder.io: Utilized for visual development and generating responsive UI components.

AI-Assisted Development: Leveraged Bolt and Builder.io for rapid UI prototyping, layout generation, and prompt-driven code scaffolding.
